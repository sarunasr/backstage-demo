# inventory-service

The inventory service keeps operational stock state aligned with incoming order demand and warehouse fulfillment signals.

## Responsibilities

- reserve inventory for confirmed orders
- reconcile stock availability across locations
- publish fulfillment readiness events

## Ownership

- owner: `team-operations`
- system: `operations-fulfillment`
- lifecycle: `production`
