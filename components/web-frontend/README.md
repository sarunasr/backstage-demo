# web-frontend

The Fry Express customer web frontend is the main digital ordering surface for consumers and support teams.

## Responsibilities

- product browsing and cart flows
- checkout and payment initiation
- order confirmation and delivery tracking

## Ownership

- owner: `team-platform`
- system: `customer-commerce`
- lifecycle: `production`
