# payments-service

The payments service handles payment transaction workflows and maintains the internal payment ledger for Fry Express Systems.

## Responsibilities

- authorize customer payment methods
- capture approved payments
- process refunds and ledger updates

## Ownership

- owner: `team-payments`
- system: `payments-ledger`
- lifecycle: `production`
