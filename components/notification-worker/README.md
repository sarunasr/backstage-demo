# notification-worker

The notification worker consumes operational events and sends customer-facing transactional communications.

## Responsibilities

- process asynchronous order and payment events
- trigger confirmation and status update notifications
- isolate delivery messaging from synchronous request paths

## Ownership

- owner: `team-operations`
- system: `operations-fulfillment`
- lifecycle: `production`
