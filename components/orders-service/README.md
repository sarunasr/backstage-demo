# orders-service

The orders service manages the order lifecycle from checkout submission through fulfillment handoff.

## Responsibilities

- create and persist customer orders
- validate order state transitions
- publish operational events for downstream workflows

## Ownership

- owner: `team-platform`
- system: `customer-commerce`
- lifecycle: `production`
