# Python FastAPI Service Starter

## Overview

This software template represents the standard bootstrap path for a new Python FastAPI service at Fry Express Systems. It captures default catalog metadata and TechDocs starter content used by platform teams.

## Intended Use

- create a new internal backend service
- generate Backstage catalog metadata up front
- provide a clean README and TechDocs baseline for delivery teams

## Ownership

- owner: Team Platform
- type: service template
